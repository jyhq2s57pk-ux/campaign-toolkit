Trading Toolkit

Quick Start & Handover Guide

1. What is it?

The Trading Toolkit is the central hub for global campaign information, activation strategies, and marketing resources.

For Viewers
Browse campaign details, calendars, and resources.
For Editors
Manage content via the Admin Panel.

2. Accessing the System

Login

Navigate to http://localhost:5173 (or your deployment URL).

Log in using your Supabase credentials.

3. Using the Toolkit

View Only The Dashboard

Upon login, you'll see the active campaign overview. Use the top navigation to explore Touchpoints, Calendar, and Resources.

Admin Editing Content

To make changes:

  1. Click the Admin toggle in the top-right corner of the navigation bar.
  2. Select the section you wish to edit from the new menu items.
  3. Changes are saved automatically or via clearly marked "Save" buttons.

Note: You need Admin permissions to see this toggle.

4. Troubleshooting

Can't log in? Contact the system administrator to verify your account has been created in Supabase.

Don't see the Admin toggle? Your account likely defaults to "Read Only". Request an upgrade from an Admin.
